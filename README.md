# VIMRC files

A bunch of configurations and plugins for vim.

## Installation
```sh
# Backup vim config folder
mv ~/.vim{,-bak}

# Clone current repo
git clone --recursive https://github.com/karbassi/vimrc.git ~/.vim
```
